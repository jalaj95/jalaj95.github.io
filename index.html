Displacement Measurement Device


DESCRIPTION:

A device was built to measure length of any object. The challenge was to design the device using first principles, so my team decided to build a device using principles of potentiometer. The device worked using the characteristic of a wire that resistance is directly proportional to the length of the wire. The device consisted of two wires and a slider which connected the two wire. Depending on the length of the object being measure, the resistance induced by the wires changed. The voltage drop caused by the wire was measured and converted to corresponding length value. An Arduino was used to read the voltage drop and interpret the voltage to length. Using statistical analysis the length values were adjusted to eliminate uncertainties.

SOFTWARE: C

HARDWARE: Mechanical Tools, Arduino Uno Microcontroller

Memory Allocation


DESCRIPTION:

Implemented a dynamic memory allocation algorithm: Half-fit. The algorithm discretized the entire pool of memory into chucks of different values and assigned each chuck into appropriate bucket, such that each bucket consisted of all chucks that fell under a specific range. When the user requested for a certain amount of memory, the algorithm would initiate the search from the bucket with all memory chucks greater than the requested value. If a bucket didn't have any free chucks, the algorithm looked in the next blocks. Once a suitable memory chuck was found, the algorithm returned the starting address of the block and assigned the memory block. If the suitable block was significantly larger than the requested block, then that block was cut into two pieces and the unused portion was reassigned to appropriate memory bucket. Additionally, when user decided to free a block of memory, the algorithm would free the chuck by putting the chuck back into memory pool and flagged it unallocated. While, the memory was being freed, the algorithm also checked for free neighboring chucks and coalesced them if possible. The algorithm was implemented on Keil board with ARM Cortex M3 processor.

SOFTWARE: C, Keil uVision4

HARDWARE: Keil Board
